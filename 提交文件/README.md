# 提交文件说明

本目录用于课程提交与 GitHub 展示，内容尽量保持“老师打开即可检查”的形式。

## 目录内容

- `源码包/`
  - 提供项目源码压缩包，便于单独下载和归档。
- `项目说明文档.md`
  - 说明项目目标、实现内容、技术结构与当前完成情况。
- `简要使用操作说明文档.md`
  - 提供最短使用路径，适合演示或快速复现。
- `构建产物/`
  - 放置 GitHub Actions 在 Windows 环境成功构建得到的 `.exe` 文件。

## 说明

- 课程原始大数据目录、临时输出目录和本地缓存不纳入 GitHub 仓库。
- `构建产物/` 下的 `.exe` 以 GitHub Actions Windows runner 产物为准，不使用本机 macOS 构建文件替代。
