# 简要使用操作说明文档

## 1. 环境准备

在项目根目录执行：

```bash
corepack pnpm install
cargo test --workspace
corepack pnpm --dir apps/desktop test -- --runInBand App.test.tsx
```

## 2. 启动桌面程序

开发模式：

```bash
corepack pnpm --dir apps/desktop tauri dev
```

## 3. 基本操作步骤

1. 点击“选择数据库目录”，加载已经构建好的 `.bin` 数据库目录。
2. 如果只有原始数据库发布包，点击“导入原始数据库目录并构建”，选择原始目录和输出目录，由程序自动构建并加载数据库。
3. 点击“选择查询文件”，加载 `.mgf` 或 `.txt` 查询文件。
4. 点击“开始批量鉴定”，等待结果返回。
5. 在“结果表”查看鉴定结果，在“运行日志”查看执行过程。
6. 如需导出数据库元数据，点击“导出数据库 CSV”。

## 4. GitHub 上的 Windows 可执行文件

仓库推送后，GitHub Actions 会在 Windows 环境构建 `gpmsdb-desktop-tauri.exe`。

- workflow 文件：`.github/workflows/build-windows-exe.yml`
- 成功后下载的 `.exe` 会放入本目录下的 `构建产物/`
