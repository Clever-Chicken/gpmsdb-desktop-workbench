# 项目说明文档

## 项目名称

GPMsDB 桌面化改造项目

## 项目目标

本项目将原始 GPMsDB 命令行工具链改造为可演示、可交互的桌面应用，完成以下目标：

1. 将 Pickle 形式原始数据库转换为适合高性能检索的二进制 mmap 数据库。
2. 使用 Rust 实现数据库读取、粗筛、重排和批量鉴定。
3. 使用 Tauri + React 构建桌面图形界面，支持数据库构建、查询文件加载、批量鉴定、结果展示和数据库 CSV 导出。

## 技术结构

- `crates/gpmsdb-builder`
  - 构建 `header.bin / mass_index.bin / genome_peaks.bin / meta.bin`
- `crates/gpmsdb-format`
  - 以内存映射方式读取数据库
- `crates/gpmsdb-engine`
  - 执行鉴定搜索与结果计算
- `apps/desktop`
  - 桌面端 GUI
